Shrimp Farm Management "ShrimpCare"

A shrimp farm management system with a focus on backend using ElysiaJS and Prisma ORM, with Supabase as the database. This system supports cultivation cycle data management, water quality monitoring, and shrimp growth calculations. The frontend is developed using Flutter for a responsive user experience.

ElysiaJS Prisma ORM Supabase Flutter

Payroll System

A payroll system application developed using CodeIgniter 4, with support for two user roles: Owner and Employee. This system includes attendance, payroll, complaints features, and an informative dashboard that facilitates real-time data monitoring. The system is designed to be efficient, user-friendly, and helps automate company administrative processes.

CodeIgniter 4 PHP MySQL

Photobox System

This project is a Photobox application developed using Laravel as the backend and admin panel, as well as Electron.js for the desktop application that handles the photo capture and transaction process. This system supports data management, transaction flow, and integration between the desktop application and backend, so the photobox operational process runs structured, efficient, and controlled.

Laravel Electron JS REST API MySQL

Blockchain-Based Diploma Recording and Verification

This project is a blockchain-based DApp for diploma recording and verification that utilizes smart contracts. The system was developed using Solidity with the Hardhat framework, with blockchain integration through Ethers.js and a React.js-based user interface. This application enables decentralized diploma data recording and a transparent, secure, and tamper-proof verification process.

Solidity Ethereum Hardhat EthersJS ReactJS
